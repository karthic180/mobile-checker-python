# Mobile Coverage Checker

This is a simple mobile coverage checker application. It allows users to check mobile network coverage by selecting a country and a mobile provider.

## Countries Supported:
- Saudi Arabia
- UAE
- France
- Germany
- Switzerland

## Providers Supported:
- **Saudi Arabia**: STC, Mobily, Zain
- **UAE**: Etisalat, Du
- **France**: Orange, SFR, Bouygues Telecom, Free Mobile
- **Germany**: Deutsche Telekom, Vodafone, O2
- **Switzerland**: Swisscom, Sunrise, Salt

## How to Use:
1. Select a country from the dropdown.
2. Select a mobile provider from the provider dropdown.
3. Click the "Check Coverage" button to view coverage information.

## Technologies Used:
- Python (Flask for the backend)
- HTML, CSS, JavaScript for the frontend
